HTML(Hyper Text Markup Language)
    -standard markup language for creating Web pages

--------------------------------------------------------------------------------
   <!DOCTYPE html>
   <html>
   <head>
   <title>Page Title</title>
   </head>
   <body>
  
   <h1>My First Heading</h1>
   <p>My first paragraph.</p>
   
   </body>
   </html> 
-------------------------------------------------------------------------------

-save the file ".html" or ".htm", it's the same shit and open it with a web browser

    <!DOCTYPE html> 
        
        -All html documents must start with a document type declaration
        -it's not case sensetive, <p> is the same as <P>, but lowercase
 
     <html> : the html document begins and ends with </html>
     <body>...</body> : the visible part
     <h1>...</h1> : h1 to h6, more and more tiny letters
     <p>...</p> : paragraphs, many as you want
     <a href="https://www.google.com">This is a link</a> : href, atribute
     <img src="michina.jpg" alt="michina.com" width="104" height="142"> 
            
        -src : the source file
        -alt : alternative text
        -width and height : attributes

    <br> : this tag define a line break, empty element without a closing tag 
        
         <p>This is a <br> paragraph with a line break.</p>
